About The Projects (2022-2023)

  • Obstacle Avoiding Robot- Made by Atish. This project avoids the obstacle in the way with the help of Ultrasoinc sensor.
  • Voice Control Robot- Made by Harshit and Kanish. This project uses a bluetooth module which takes commands from a device connected to it and then the robot follows the command.
  • Drone- Made by Vethapriyan. This project uses a radio remote controller to control the Drone.
  • Human Following Robot- This is made by D. Manish, Roshan, Shrusti and Darshan. This project uses 3 sensors to give the most accurate input to the processor so it works.
  • Smart Parking- This is made by Manigandan and Atish. It uses sensors to detect a car to open the gate of the parking.
  • Blind Stick- This project is made by Aakruti, Jayantika and Srimathi. It is made for specialy blind people. It uses a ultrasoinc sensor to detect if there is anything near which can harm the person.
  • Blink sensor- It is again made by Aakruti, Jayantika and Srimati. This is made for the people who drive in night. With sensor it senses if there eyes are closed for more than 2 seconds and buzzes the buzzer.
  • Wall-E Robot- This is made by Surya and Athitiya. It uses a simple gear motor to make the robot walk.
